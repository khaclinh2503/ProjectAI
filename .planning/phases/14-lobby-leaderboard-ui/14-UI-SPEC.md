## Spacing Scale

Declared values (must be multiples of 4). All values are Cocos canvas units.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Cell gap between grid cells (existing: CELL_GAP=4); inline icon gaps |
| sm | 8px | Label-to-label vertical gap within a section |
| md | 16px | Default element spacing; padding inside overlay panels |
| lg | 24px | Gap between lobby sections (title → greeting → buttons) |
| xl | 32px | Vertical margin between button group and screen edge |
| 2xl | 48px | Lobby button height (touch target; also satisfies 44px minimum) |
| 3xl | 64px | Top padding from safe area to title label |

Exceptions:
- Lobby button touch target: minimum 44px height → use 48px (2xl) to satisfy accessibility and round to 8-point scale.
- LeaderboardScene row height: 40px (within 8-point scale, acceptable for list density at 10 rows).
- Toast label vertical offset above button group: 16px (md) above the bottom button.

Source: existing CELL_GAP=4 and CELL_SIZE=68px in GridRenderer.ts confirm 4-unit base; CONTEXT.md D-10 (vertical stack layout).

## Typography

All sizes are Cocos Label `fontSize` values (canvas units). No CSS.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Display | 40px | Bold (700) | 1.2 | LobbyScene title "BLOOM TAP"; LeaderboardScene title "BẢNG XẾP HẠNG" |
| Heading | 24px | SemiBold (600) | 1.3 | Name overlay title ("Nhập tên của bạn"); section headers |
| Body | 18px | Regular (400) | 1.5 | Greeting label "Xin chào, [name] 🌸"; leaderboard row entries (rank, name, score); toast label "Sắp ra mắt!" |
| Label | 20px | SemiBold (600) | 1.2 | Lobby button text; "Bắt Đầu" confirm button; "Back" button on LeaderboardScene |

Exactly 4 sizes declared. Exactly 2 weights used (Regular 400, SemiBold/Bold 600–700).

Source: CONTEXT.md D-10 (title "BLOOM TAP"), D-09 (empty state label), D-03 (EditBox), D-08 (3-column rows); discretion on exact sizes.

## Color

All colors are Cocos `Color(r, g, b, a)` values. Hex equivalents provided for reference.

| Role | Cocos Color | Hex Equivalent | Usage |
|------|------------|----------------|-------|
| Dominant (60%) | Color(30, 20, 45, 255) | #1E142D | Scene background — dark purple-black; matches existing GameScene dark aesthetic |
| Secondary (30%) | Color(55, 40, 75, 255) | #37284B | Overlay panel background (name-input modal), leaderboard row alternating fill, button container area |
| Accent (10%) | Color(255, 220, 60, 255) | #FFDC3C | Reserved for: primary CTA "Chơi Ngay" button fill; leaderboard #1 rank label; enabled "Bắt Đầu" button fill |
| Destructive | Color(220, 50, 50, 255) | #DC3232 | Not used in this phase — no destructive actions |

Accent reserved for: **"Chơi Ngay" button fill, "Bắt Đầu" confirm button fill (enabled state only), top-rank (#1) text highlight in LeaderboardScene.** Never applied to disabled buttons, toast labels, or secondary nav buttons.

Color notes:
- **Name overlay dim**: semi-transparent dark layer behind modal — Color(0, 0, 0, 160) alpha=160 (~63% opacity). Source: CONTEXT.md D-05 ("lobby content visible but dimmed").
- **Disabled button ("Bắt Đầu" with 0 chars)**: Color(100, 90, 115, 180) — grayed, interactable=false. Source: CONTEXT.md D-03.
- **Secondary nav buttons** (Vườn Hoa, Túi Đồ, BXH, Setting): Color(70, 55, 95, 255) — mid-purple, no accent fill.
- **Toast label text**: Color(255, 255, 255, 255) — white text on semi-transparent dark panel (Color(30, 20, 45, 200)).
- **Existing flash colors** reused as-is: CORRECT_FLASH_YELLOW (255, 220, 60), WRONG_FLASH_COLOR (220, 50, 50) — no changes.

Source: FlowerColors.ts (CORRECT_FLASH_YELLOW matches accent); CONTEXT.md D-11 ("solid background color consistent with existing scenes"); GameController.ts TIMER_COLOR_NORMAL = white.

## Interaction States

### LobbyScene Button States

| Button | Normal | Disabled | On Tap |
|--------|--------|----------|--------|
| Chơi Ngay | Accent fill (#FFDC3C), Label white | — | `director.loadScene('GameScene')` |
| Vườn Hoa | Secondary fill | — | Show toast "Sắp ra mắt!" |
| Túi Đồ | Secondary fill | — | Show toast "Sắp ra mắt!" |
| BXH | Secondary fill | — | `director.loadScene('LeaderboardScene')` |
| Setting | Secondary fill | — | Show toast "Sắp ra mắt!" |

### Name Input Overlay States

| State | Condition | Visual |
|-------|-----------|--------|
| Overlay visible | `getPlayerName() === null` on LobbyScene load | active=true, DimBackground shows |
| Bắt Đầu disabled | EditBox length === 0 | grayed Color(100,90,115,180), interactable=false |
| Bắt Đầu enabled | EditBox length >= 1 | accent fill Color(255,220,60,255), interactable=true |
| Confirming | Tap Bắt Đầu | Call `setPlayerName()`, UIOpacity tween 255→0 over 0.3s, then active=false |
| Lobby active | After overlay fade-out | GreetingLabel updates to "Xin chào, [name] 🌸" |

Source: CONTEXT.md D-01 through D-05, D-12.

### Toast Animation

| Step | Value |
|------|-------|
| Trigger | Tap Vườn Hoa / Túi Đồ / Setting |
| Show animation | UIOpacity tween 0→255 over 0.2s |
| Hold duration | 1.5s |
| Hide animation | UIOpacity tween 255→0 over 0.2s |
| Interrupt behavior | New tap restarts tween immediately (kill previous, restart from step 1) |
| Blocks UI | No — non-modal |

Source: CONTEXT.md D-13 through D-15; Claude's Discretion (0.2s fade matches existing juice style).

### LeaderboardScene Row Display

| Condition | Behavior |
|-----------|----------|
| Entries exist (1–10) | Show RowNode[0..N-1] active=true; fill RankLabel "#N", NameLabel, ScoreLabel |
| Entries < 10 | RowNode[N..9] active=false |
| No entries | All RowNodes active=false; EmptyStateLabel active=true |
| On scene load | `LeaderboardController.onLoad()` calls `getEntries()` and re-renders — always fresh |

Source: CONTEXT.md D-08, D-09; code_context (refresh on onLoad).
